Recursividad en Ciencias Computacionales

La recursividad es un pilar fundamental que permite resolver problemas complejos dividiéndolos en versiones más simples de sí mismos.

Importancia

La recursividad es fundamental para estructurar algoritmos eficientes, simplificar soluciones a problemas inherentemente recursivos y profundizar la comprensión de procesos computacionales de "divide y vencerás".

Algoritmo Factorial

El factorial de un número n, denotado n!, es el producto de todos los enteros positivos desde 1 hasta n. Se define recursivamente como:

factorial(n) = n * factorial(n - 1), con factorial(0) = 1.

Diagrama de llamadas recursivas para el cálculo factorial

Ejemplo en JavaScript

function factorial(n) {
    if (n === 0) return 1;
    return n * factorial(n - 1);
}

Algoritmo Fibonacci

La sucesión de Fibonacci es una serie donde cada número es la suma de los dos anteriores, comenzando con 0 y 1. Se define recursivamente:

fibonacci(n) = fibonacci(n-1) + fibonacci(n-2) con fibonacci(0) = 0 y fibonacci(1) = 1.

Árbol de llamadas recursivas para fibonacci(4)

Ejemplo en JavaScript

function fibonacci(n) {
    if (n === 0) return 0;
    if (n === 1) return 1;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

Torres de Hanói

Las Torres de Hanói es un rompecabezas donde se deben mover discos entre postes con la regla de no poner disco más grande sobre uno más pequeño. Se resuelve recursivamente:

Visualización de Torres de Hanói con 3 discos

Ejemplo en JavaScript

function hanoi(n, origen, destino, auxiliar) {
    if (n === 1) {
        console.log(`Mover disco 1 de ${origen} a ${destino}`);
        return;
    }
    hanoi(n - 1, origen, auxiliar, destino);
    console.log(`Mover disco ${n} de ${origen} a ${destino}`);
    hanoi(n - 1, auxiliar, destino, origen);
}

Fractales

Los fractales son figuras geométricas que se repiten a diferentes escalas, creadas mediante procesos recursivos, mostrando auto-similitud.

Triángulo de Sierpinski, un fractal generado recursivamente

Ejemplo simple en JavaScript

function sierpinski(depth) {
    if (depth === 0) {
        // Dibujar triángulo base
        return;
    }
    // Llamadas recursivas para subdividir
    sierpinski(depth - 1);
    sierpinski(depth - 1);
    sierpinski(depth - 1);
}

Conclusión sobre la Recursividad

La recursividad es una técnica esencial en ciencias computacionales para resolver problemas complejos dividiéndolos en problemas más simples. Facilita soluciones elegantes y comprensibles, aunque requiere atención en la gestión de memoria y optimización.

Los algoritmos recursivos como Factorial, Fibonacci, Torres de Hanói y fractales representan áreas donde esta técnica es indispensable, con aplicaciones en matemáticas, algoritmos y gráficos.

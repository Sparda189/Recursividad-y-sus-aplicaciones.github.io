Recursividad en Algoritmos

Investigación y ejemplos prácticos sobre algoritmos recursivos en ciencias computacionales

Introducción

La recursividad es una técnica de programación en la que una función se llama a sí misma para resolver un problema dividiéndolo en subproblemas más simples. Es esencial en ciencias computacionales porque permite resolver problemas complejos con soluciones elegantes y claras.

Esta página presenta los fundamentos y ejemplos gráficos de cuatro algoritmos clásicos que emplean recursividad: el cálculo factorial, la sucesión de Fibonacci, las Torres de Hanói y los fractales.

Algoritmo Factorial

El factorial de un número n (denotado como n!) es el producto de todos los enteros positivos desde 1 hasta n. Se define recursivamente como:

factorial(n) = n * factorial(n - 1), con factorial(0) = 1.

function factorial(n) {
    if (n === 0) return 1;
    return n * factorial(n - 1);
}

Algoritmo Fibonacci

La sucesión de Fibonacci es una serie donde cada número es la suma de los dos anteriores, comenzando con 0 y 1. La definición recursiva es:

fibonacci(n) = fibonacci(n-1) + fibonacci(n-2) con fibonacci(0) = 0 y fibonacci(1) = 1.

function fibonacci(n) {
    if (n === 0) return 0;
    if (n === 1) return 1;
    return fibonacci(n - 1) + fibonacci(n - 2);
}

Torres de Hanói

Las Torres de Hanói es un rompecabezas con tres postes y varios discos de distinto tamaño. El objetivo es mover todos los discos del poste inicial al poste destino, siguiendo reglas:

  • Solo se puede mover un disco a la vez.
  • No se puede colocar un disco más grande sobre uno más pequeño.

El algoritmo recursivo es:

function hanoi(n, origen, destino, auxiliar) {
    if (n === 1) {
        console.log(`Mover disco 1 de ${origen} a ${destino}`);
        return;
    }
    hanoi(n - 1, origen, auxiliar, destino);
    console.log(`Mover disco ${n} de ${origen} a ${destino}`);
    hanoi(n - 1, auxiliar, destino, origen);
}

Fractales

Los fractales son figuras geométricas que se repiten a diferentes escalas, obtenidas con procesos recursivos. Suelen presentar autosemejanza.

Ejemplo: Triángulo de Sierpinski, se obtiene dividiendo un triángulo en cuatro triángulos más pequeños y quitando el central recursivamente.

function sierpinski(depth) {
    if (depth === 0) {
        // Dibujar triángulo base
        return;
    }
    sierpinski(depth - 1);
    sierpinski(depth - 1);
    sierpinski(depth - 1);
}

Conclusión

La recursividad es una herramienta poderosa en ciencias computacionales que permite resolver problemas complejos de manera clara y elegante. Los algoritmos presentados demuestran cómo la recursión reduce problemas a subproblemas más simples y facilita soluciones estructuradas.

Comprender y aplicar correctamente la recursividad es fundamental para desarrollar algoritmos eficientes y para la enseñanza en programación y matemáticas.
